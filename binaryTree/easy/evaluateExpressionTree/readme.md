# Evaluate binary expression trees
This  project  evaluates binary expression trees.  <br />
The leaves of the trees represent values (greater than or equal to 0) and the non-leaf nodes represent operations  <br />
 * addition `-1` <br >
 * subtraction `-2` <br >
 * division `-3` <br >
 * multiplication `-4` <br >
